## Requirements
- Node.js installed

## Steps

1. Install the Project Dependencies
```bash
npm install
```
2. Create a .env File
```env
REACT_APP_API_URL=http://localhost:8080/api/v1
REACT_APP_OAUTH_CLIENT_ID=
```
3. Start the Server
```bash
npm start
```
